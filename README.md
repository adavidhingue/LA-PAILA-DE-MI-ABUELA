# LA-PAILA-DE-MI-ABUELA
La Paila de Mi Abuela es un sitio web diseñado para promocionar un restaurante de comida típica. Permite a los clientes consultar el menú, conocer promociones, ver galerías de platos tradicionales, realizar reservas y contactar al restaurante de forma rápida y sencilla, mejorando la experiencia del usuario y la presencia digital del negocio.
